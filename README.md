# dev-clojure-cli-influxdb-simple

## Description
Creates a small database table
called `dog` and populates with
hql.

## Tech stack
- clojure

## Docker stack
- clojure
- tutun/influxdb

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
